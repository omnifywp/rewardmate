*** Omnify Customer Rewards WooCommerce Changelog ***

2026-07-21 - version 1.4.0
* Rebranded product name to Omnify Customer Rewards WooCommerce (admin menu remains RewardMate).
* Renamed main plugin and asset files to omnify-customer-rewards*.
* REST/webhook headers use X-Omnify-Customer-Rewards-* with legacy X-RewardMate-* support.


2026-04-30 - version 1.3.11
* Fixed expanded filter panels overflowing admin cards on ledger, analytics, and tools screens.
* Improved adjust-points form alignment, settings builder widths, campaign selector heights, and customer history table readability.
* Strengthened customer wallet filter button styling so it stays consistent with Omnify Customer Rewards buttons.

2026-04-30 - version 1.3.10
* Added dynamic category and product selectors for multiplier rules, exclusions, and scheduled campaigns.
* Preserved backward-compatible saved formats while adding stricter sanitization for multiplier, CSV ID, campaign, and tier settings.
* Polished admin filter buttons, CSV import buttons, wallet balance spacing, point history tables, and duplicate daily check-in messaging.

2026-04-30 - version 1.3.9
* Rebuilt loyalty tiers as dynamic admin-configurable rules instead of fixed Silver/Gold/Platinum fields.
* Updated tier assignment, admin filters, CSV exports, REST responses, wallet displays, and tier-upgrade emails to use custom tier labels.

2026-04-30 - version 1.3.8
* Added Omnify Customer Rewards date fields to user profiles for birthday, account anniversary, and first purchase anniversary rewards.
* Updated milestone reward automation to use the profile account anniversary date when provided.

2026-04-30 - version 1.3.7

2026-04-30 - version 1.3.6

2026-04-30 - version 1.3.5

2026-04-30 - version 1.3.4

2026-04-29 - version 1.3.3

2026-04-29 - version 1.3.2

2026-04-29 - version 1.3.0

* Added CSV import/export for user balances, point history, ledger entries, and liability reports.
* Added bulk points adjustment for selected users, roles, loyalty tiers, and customers who purchased a product.
* Added birthday, account anniversary, and first purchase anniversary rewards.
* Added separate email notification toggles and templates for loyalty events.
* Added secure REST API endpoints and outbound webhooks for point balance integrations.
* Added Fraud Review dashboard for suspicious activity flags and manual review status.
* Added a checkout redemption slider with server-side caps for balance, minimum, step size, coupons, and max redemption percent.

* Added category/product multipliers, exclusions, and campaign scheduler.
